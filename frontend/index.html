學術思辨：單熱編碼 (OHE) 與 貝氏目標編碼 (BTE) 的冗餘度論證

在許多機器學習項目中，開發者常有「越複雜的演算法越好」的盲點。在此專案中，我們同時納入了「單熱編碼 (One-Hot Encoding)」與「貝氏目標編碼 (Beta Target Encoding)」。

🔍 本數據集上的設計冗餘 (Redundancy)

本數據集中的類別型特徵為「州別 (State)」，僅有 3 個類別：加州 (California)、佛州 (Florida)、紐約州 (New York)。

對於基數為 3 ($K=3$) 的變數，使用單熱編碼只需生成 2 個虛擬變數（丟棄加州作基準），即可完美且無損地表徵地理區位。在此情境下，使用複雜的貝氏目標編碼（正規化、先驗參數、後驗更新、提取均值與方差）在數學上是完全冗餘且過度設計的（Overkill）。

💡 概念驗證與工業級價值 (PoC Value)

既然如此，為什麼還要設計與保留 BTE 呢？

我們將其定位為一個 **概念驗證 (Proof of Concept, PoC)**。在實際工業級應用中，我們經常遇到「高基數特徵」(High Cardinality)，例如：郵遞區號 (500+類別)、商品類別 (1,000+類別)。若用單熱編碼，會造成特徵空間急劇膨脹，特徵矩陣極度稀疏，進而引發 OLS 迴歸維度災難與嚴重過度擬合。

此時，BTE 可以優雅地將高基數欄位壓縮成均值與方差 **2個實數欄位**，並通過平滑係數 $m$ 調節防噪與抗漏失能力。本專案向您完整展示了這套貝氏特徵工程的實作流程。

Phase 1: 商業理解 02 / 08

早期風險投資的決策痛點

📌 投資難題

  • 新創公司財務歷史數據稀疏，傳統估值方法失效。
  • 難以量化不同運營支出項目（研發、行政、行銷）對淨利潤的邊際貢獻。
  • 設立州別（地理位置）是否有潛在的稅收與政策效應？

🎯 專案任務

透過建立「白盒化」線性迴歸模型，量化預算邊際回報率，並給出帶有信賴區間的利潤估計，輔助 VC 科學化決策。

Phase 2: 數據理解 03 / 08

數據探索與相關性矩陣

Phase 3: 數據準備與編碼 04 / 08

類別型變數編碼與完全共線性

1. One-Hot Encoding & 虛擬變數陷阱

對於 State (CA, FL, NY) 進行單熱編碼，若將所有州與常數項一同放入模型，會導致 $\mathbf{v}_{CA} + \mathbf{v}_{FL} + \mathbf{v}_{NY} = \mathbf{1}$。

列向量完全線性相關，使得矩陣 $X^T X$ 無法求逆（即其逆矩陣 $(X^T X)^{-1}$ 不存在）！

解決方案：丟棄首個州別 (California) 作為基準類別。

⚠️ 貝氏編碼冗餘警告：

在此數據集上 ($K=3$) 實作貝氏目標編碼是「冗餘的」。單熱編碼完全足夠。

本專案保留此設計是用作 **概念驗證 (PoC)**，展示其在處理工業級 **「高基數特徵」** 時的壓縮與防過度擬合能力。

Phase 3: Beta 目標編碼數學 05 / 08

迴歸任務下的 Beta Target Encoding

我們將分類任務的 Beta 目標編碼推廣到連續迴歸中，步驟如下：

1
目標值歸一化：
將 Profit $y$ 歸一化到 $[0, 1]$ 區間，使其符合 Beta 先驗。
2
先驗與後驗更新：
計算全域均值 $p$，以平滑係數 $m$ 決定先驗。結合觀測更新後驗期望：
E[y_{norm} | c] = (S_c + m·p) / (n_c + m)
3
後驗均值與方差雙特徵：
同時提取代表不確定性的方差：
Var(y_{norm} | c) = (\alpha·\beta) / [(\alpha+\beta)^2(\alpha+\beta+1)]
並逆變換回美元尺度。
Phase 4: 模型建構 (Modeling) 06 / 08

最小平方法 (OLS) 解析求解與推論

📐 OLS 矩陣偏微分證明

為最小化殘差平方和 $S(\boldsymbol{\beta}) = (\mathbf{y} - X\boldsymbol{\beta})^T (\mathbf{y} - X\boldsymbol{\beta})$，對其求一階導數並令其為零：

-2 X^T \mathbf{y} + 2 X^T X \hat{\boldsymbol{\beta}} = \mathbf{0}

在 $X^T X$ 可逆的假設下，得到唯一解析解：

\hat{\boldsymbol{\beta}} = (X^T X)^{-1} X^T \mathbf{y}

📊 白盒推論 (Inference)

  • 計算殘差無偏方差 $s^2 = \mathbf{e}^T\mathbf{e} / (n - k - 1)$。
  • 係數共變異數矩陣 $Var(\hat{\boldsymbol{\beta}}) = s^2 (X^T X)^{-1}$，其對角線平方根為標準誤差 $SE$。
  • 計算 $t = \hat{\beta} / SE(\hat{\beta})$ 以評估 $p$ 值，確定統計顯著性。
